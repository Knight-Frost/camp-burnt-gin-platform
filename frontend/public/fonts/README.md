# Landing Page Fonts

## Required Fonts

Download WOFF2 files from Google Fonts:

### Fraunces
- Weight: 700 (Bold)
- Weight: 900 (Black)
- URL: https://fonts.google.com/specimen/Fraunces
- Download WOFF2 files and place in this directory:
  - Fraunces-Bold.woff2
  - Fraunces-Black.woff2

### Inter
- Weight: 400 (Regular)
- Weight: 500 (Medium)
- URL: https://fonts.google.com/specimen/Inter
- Download WOFF2 files and place in this directory:
  - Inter-Regular.woff2
  - Inter-Medium.woff2

## Temporary Fallback

Until fonts are downloaded, using Google Fonts CSS import as fallback.
